# ✨ Image Processor Pro

تطبيق معالجة صور احترافي باستخدام **Python** و **OpenCV**.

### المميزات
- تحميل الصور بسهولة
- تحويل الصورة إلى رمادي
- تطبيق Gaussian Blur
- كشف الحواف (Canny Edge Detection)
- تحسين التباين والسطوع والحدة
- واجهة حديثة Dark Mode
- حفظ الصور المعالجة

### كيفية التشغيل

```bash
# 1. تثبيت المكتبات
pip install -r requirements.txt

# 2. تشغيل التطبيق
python image_processor.py
